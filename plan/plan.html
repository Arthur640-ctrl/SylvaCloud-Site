Trouvez le plan qu'il vous faut

Trouvez le plan idéal pour vos besoins et vos envies. Si vous augmentez de plan, vos donnés seront immediatement transferés

Quel plan est le plus adapté pour moi ?

Si vous utilisez moins de 15 Go de stockage, notre plan Gratuit est parfait. Pour une utilisation plus régulière avec jusqu'à 50 Go, le plan Essentiel est idéal. Si vous avez besoin de plus de stockage et de fonctionnalités avancées, le plan Pro à 200 Go sera le mieux adapté. Les entreprises peuvent opter pour notre plan Entreprise avec des options de stockage personnalisées et un support dédié.

Est ce qu'un payement est necessaire pour le plan gratuit ?

Non, le plan gratuit offre 15 Go de stockage sans frais. Aucun paiement n'est nécessaire pour ce plan.

Est ce que le plan gratuit l'est à vie ?

Oui, le plan gratuit est gratuit à vie, avec 15 Go de stockage. Cependant, des options payantes sont disponibles si vous avez besoin de plus de stockage ou de fonctionnalités avancées.
